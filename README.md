
# Portfolio

Este repositorio contiene el código fuente de mi portfolio personal desarrollado en React.

En este proyecto podrás explorar en detalle la implementación completa de la aplicación, incluyendo la estructura del código, la organización de componentes y las decisiones técnicas adoptadas durante el desarrollo.

Tecnologías utilizadas
React
Tailwind CSS
Backend ligero en PHP

El desarrollo de la web se ha realizado principalmente a partir de conocimientos propios, complementados con el uso puntual de herramientas de IA (Vibe Coding) para pruebas y validación de funcionalidades.

Objetivo

Este proyecto tiene como finalidad mostrar mi forma de trabajar y permitir evaluar mis competencias en:

JavaScript
React
Estructuración de proyectos frontend
Integración básica con backend

Si quieres, puedo adaptarlo a un tono más informal, más técnico (tipo documentación de ingeniería) o más orientado a reclutadores.


## Tech Stack

**Client:** React, TailwindCSS

**Server:** PHP
